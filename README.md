# 🧮 Калькулятор на Python (CustomTkinter)

## Опис

Цей проєкт — простий графічний калькулятор, створений за допомогою **Python** та бібліотеки **CustomTkinter**.

Програма має зручне вікно з кнопками для введення чисел і виконання основних математичних операцій.

## Можливості

* Введення чисел за допомогою кнопок
* Виконання базових операцій:

  * додавання `+`
  * віднімання `-`
  * множення `*`
  * ділення `/`
* Кнопка `=` для отримання результату
* Кнопка очищення поля
* Захист від введення декількох операторів підряд

## Використані технології

* Python 3
* CustomTkinter
* Tkinter Grid Layout

## Встановлення

1. Склонуйте репозиторій:

```bash
git clone https://github.com/your-username/calculator.git
```

2. Встановіть бібліотеку CustomTkinter:

```bash
pip install customtkinter
```

3. Запустіть програму:

```bash
python calculator.py
```

## Як користуватися

1. Натискайте на кнопки з цифрами для введення числа.
2. Оберіть потрібну математичну дію.
3. Натисніть `=` для отримання результату.
4. Використовуйте кнопку очищення, щоб видалити введені дані.
